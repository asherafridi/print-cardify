# PrintCardify

PrintCardify is a lightweight web application designed for printing cards with content on both sides. This application ensures that the content on both sides of the card is perfectly aligned when printed from the same location on a page.

## Features

- **Dual-Sided Printing**: Easily print cards with content on both the front and back sides.
- **Precise Alignment**: Ensure accurate alignment of content on both sides for a professional finish.
- **User-Friendly Interface**: Intuitive interface for a seamless user experience.
- **Customizable Printing Options**: Adjust printing settings for various card sizes and orientations.
